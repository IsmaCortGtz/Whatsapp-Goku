
# Add your sources

You need to put here your sources, add the song in a `sound.mp3` file, and your images in a folder named "images" with this format `ssj1.jpg`, `ssj2.jpg`, ect.

## Example tree

```
.
├── sources
│   ├── sound.mp3
│   └── images
│       ├── ssj1.jpg
│       ├── ssj2.jpg
│       ├── ssj3.jpg
│       ├── ...
│       └── ssj20.jpg
└── ...
```
